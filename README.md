## 📌 Project Overview

The **AI-Based Tactical Decision Support System** is an intelligent web-based platform that leverages Artificial Intelligence and Machine Learning to support tactical and strategic decision-making processes.

The system integrates battlefield simulation, satellite-based mapping, and AI-driven analysis to provide actionable insights for mission planning and operational assessment. By combining real-time visualization with intelligent recommendations, the platform enhances situational awareness and decision accuracy.

This project demonstrates the practical application of AI in defense-oriented systems and strategic planning environments.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Shindesuraj-001/Ai-Based-Tactical-decision_support-system.git
```

### 2️⃣ Navigate to the Project Directory

```bash
cd Ai-Based-Tactical-decision_support-system
```

### 3️⃣ Create a Virtual Environment (Recommended)

```bash
python -m venv venv
```

### 4️⃣ Activate the Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### 5️⃣ Install Required Dependencies

```bash
pip install -r requirements.txt
```

### 6️⃣ Run the Flask Application

```bash
python app.py
```

### 7️⃣ Open in Browser

```text
http://127.0.0.1:5000
```

or

```text
http://localhost:5000
```
